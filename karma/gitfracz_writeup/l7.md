**Level Description ->** Use git stashing
```
git stash 
nano bug.txt #make the changes
git add bug.txt
git commit -sS -m "make bugfix"
git stash pop
nano bug.txt #make final changes
git add bug.txt
git commit -sS -m "final change commit"
git verify
```
