# Git Excercises


### Master

```sh
git commit
```

Used to create a commit.

```sh
git push
```

Used to push all commits made.

### Commit-One-File

```sh
git add <file>
```

Registers an untracked file.

### Commit-One-File-Staged

```sh
git restore --staged <file>
```

Unstages a file so it is not committed.
